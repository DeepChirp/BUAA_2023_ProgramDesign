# Review - 期末考试专项复习特别辑

## 比赛简介

1. 本比赛**不计入课程成绩**，专门为同学们期末复习提供练习。
2. 助教头子特意将题目进行了归类和难度评定，见下方表格，请同学们根据自身情况挑选合适的题目来做。
3. “《C语言程序设计实训教程》练习”系列题目更偏基础，本比赛的题目更加综合。
4. 本比赛的题解将会在11月20日发布，届时同学们可以参照题解，查漏补缺。

## 题目设置

+ 代码基础题11道
+ 复杂模拟题7道（码量均较大，可以跳过）
+ `qsort`专项练习题4道（重点）
+ 递归专项练习5道（难点）
+ 思维题（脑际急转弯or算法题）12道
